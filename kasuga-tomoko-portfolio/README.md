# 沈金師・春日友子 ポートフォリオサイト

芸術関係者・マスメディア(露出機会・認知度アップにつながる方々)を対象とした、
沈金師・春日友子氏のポートフォリオサイト。プレスキットとして機能することを重視した構成です。

ビルド不要の静的サイト一式です。このフォルダをそのままWebサーバーのドキュメントルート
(または任意のサブディレクトリ)に配置すれば動作します。

## 写真の差し替え手順

すべての作品画像は `<img src="….jpg" onerror="→ ….svg">` 方式です。
**同名の .jpg ファイルを置くだけで、HTMLを編集せずに実写真に切り替わります。**

| 置き場所 | 内容 | 推奨サイズ |
|---|---|---|
| `images/works/a-man.jpg` | 《A MAN》 | 縦位置 800×1000px 以上 |
| `images/works/hannya-shingyo.jpg` | 《輪島塗朱塗大盤 般若心経》 | 同上 |
| `images/works/onore.jpg` | 《己》 | 同上 |
| `images/works/kirameki.jpg` | 《煌〜きらめき〜》 | 同上 |
| `images/works/yu.jpg` | 《優》 | 同上 |
| `images/works/yunagi.jpg` | 《夕凪》 | 同上 |
| `images/works/ichiyo-boketsu.jpg` | 《「一葉女子の墓」一部模写》 | 同上 |
| `images/works/acrylic-chinkin.jpg` | アクリル沈金キーホルダー | 同上 |
| `images/portrait.jpg` | 作家近影 | 同上 |
| `images/ogp.png` | SNSシェア用画像(**上書き**) | 1200×630px |

- 実写真を置いたら、対応する `<figcaption>イメージ画像(写真差し替え予定)</figcaption>` の
  行を削除し、`alt` 属性の「(写真準備中)」を撮影者クレジット等に書き換えてください。
- 縦横比はCSS(`aspect-ratio: 4/5`)でトリミング表示されます。比率の異なる写真は
  `style.css` の `.work-media img` を調整してください。

## ローカルでの確認方法

```bash
# このフォルダで
python3 -m http.server 8000
# → http://localhost:8000/ を開く
```

すべてのリンク・アセットは相対パスのため、サブディレクトリ配下
(例: `https://example.com/kasuga-tomoko-portfolio/`)でもドメイン直下でも動作します。

## デプロイ

- **独自ドメイン(推奨: chinkinshi-yuko.art 等)**: このフォルダの中身をドキュメントルートに配置。
- **GitHub Pages**: 専用リポジトリを作りこのフォルダの中身をプッシュ、Pages を有効化。
  独自ドメインを使う場合は `CNAME` ファイルを追加。
- デプロイ先が決まったら、各HTMLの `canonical` / `og:url` / JSON-LD の絶対URL
  (現在は `https://chinkinshi-yuko.art/` を仮置き)を実際の公開URLに合わせてください。

## 構成メモ

- 配色: 漆黒(#14100C)×生成り(#F2EBDC)×金(#C9A24B)。朱(#B3341F)は装飾・大型文字限定(WCAG AA)
- 書体: Shippori Mincho(見出し・引用)+ Noto Sans JP(本文)
- `profile.html` は `@media print` で印刷するとそのままプレスキットになります
- JS(`js/main.js`)はナビ開閉と表示演出のみ。無効環境でも全コンテンツ閲覧可能
- 構造化データ: Person / ExhibitionEvent / VisualArtwork(JSON-LD)
