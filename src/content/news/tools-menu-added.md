## 掲載ツール

### [PDF請求書のAI自動読取・記帳システム](/tools/invoice)

請求書PDFをフォルダに入れるだけで、AIが読み取り・記帳・整理を全自動化するシステムです。電子帳簿保存法・インボイス制度への対応も自動で行います。

- **技術**：Google Apps Script / Claude Vision API / Google Drive
- **費用感**：5〜10万円規模

### [研修交流会チーム自動編成システム](/tools/team-assignment)

40名超の研修・交流会で、8項目にのぼる複雑な制約条件をすべて満たすチーム振り分けを数秒で自動生成するシステムです。PowerPointスライドの自動出力にも対応しています。

- **技術**：Python / PowerPoint自動生成
- **費用感**：5～10万円規模
