# 沈金師・春日友子「暮らしの道具」ランディングページ

沈金をほどこした箸・漆器などの日用品の販売と、金継ぎ（欠けた・割れた器の修理）の
受付につなげる、1枚もののランディングページです。

アーティストとしての作品サイト（`kasuga-yuko-portfolio/`、別ブランチ
`claude/kasuga-tomoko-portfolio-jo45hw` で制作）とは別の、
「暮らしの雑貨屋さん・器のお店」としての入り口を担います。

- **単一HTMLファイル**（`index.html`）にCSS/JSを内包。外部依存はGoogle Fontsのみ
- スマホファーストのレスポンシブ。ナビはハンバーガーメニュー、各セクションへスムーススクロール
- サブドメイン（例: `utsuwa.chinkinshi-yuko.art`）でもサブディレクトリでも、
  このフォルダをそのまま置けば動作します

## ローカルでの確認方法

```bash
# このフォルダで
python3 -m http.server 8000
# → http://localhost:8000/ を開く
```

## 記載内容の根拠（一次情報調査レポートに基づく）

コピーの事実部分は、調査レポート「和食器・漆器の『良さ』とお手入れLP向け
一次情報調査レポート」の推奨に沿って、断定を避けた表現にしています。

- **抗菌性**: 岩手県工業技術センター「漆塗膜の抗菌性の検証」（研究報告第16号、2009年）
  を出典として脚注に明記。「99%除菌」等の断定表現は不使用。
  https://www2.pref.iwate.jp/~kiri/study/report/2008/pdf/H20_12.pdf
- **お手入れ**: 消費者庁「家庭用品品質表示法」の漆器の取扱い表示基準
  （電子レンジ不可・食洗機不可・直射日光回避・たわし/磨き粉不可）に準拠。
  https://www.caa.go.jp/policies/policy/representation/household_goods/guide/zakka/zakka_19.html
- **保温性・軽さ・口当たり**: 数値の断定はせず、生活実感の言葉で記述
  （政府広報「Highlighting Japan」の研究者見解の範囲内）。
- **「japan＝漆器」**: 「かつてヨーロッパで漆器全般を指す語として使われた」という
  歴史的文脈の記述に限定。
- **漆かぶれ・金継ぎの安全性**: 「しっかりと硬化させて仕上げたもの」という
  条件付きの表現に留め、無条件の安全断定はしていません。
